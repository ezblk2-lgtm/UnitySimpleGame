# Triffic Runner

[Русская версия ниже/English version below]

## ru Русская версия

Простая 3D игра в жанре "бесконечный бегун" на Unity. Игрок управляет машиной, движется по бесконечно генерируемой трассе и уворачивается от встречных автомобилей.

### Особенности

- Бесконечная генерация дороги с препятствиями
- Система счета (чем дальше — тем больше очков)
- Главное меню и меню настроек
- Настройка разрешения, полноэкранного режима и громкости
- Звук двигателя с изменением тона от скорости
- Сохранение настроек через PlayerPrefs

### Технологии

- Unity 6 (6000.0.xx)
- Input System Package
- C#

### Управление

 W - Ускорение  
 S - Торможение   
 A - Поворот влево   
 D - Поворот вправо   

### Запуск проекта

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ezblk2-lgtm/UnitySimpleGame.git
   ```
2. Откройте проект в Unity Hub (версия Unity 6000.0.xx или выше)

3. Убедитесь, что установлены пакеты: Input System

4. Откройте сцену `Assets/Scenes/MenuScene.unity`

5. Нажмите Play или соберите проект через File → Build Settings

## en English version

A simple 3D endless runner game made in Unity. The player controls a car, drives along an infinitely generated road, and dodges oncoming traffic.

### Features

- Infinite road generation with obstacles
- Scoring system (the further you go, the more points you get)
- Main menu and settings menu
- Resolution, fullscreen mode, and volume settings
- Engine sound with pitch shifting based on speed
- Settings saved via PlayerPrefs

### Tech Stack

- Unity 6 (6000.0.xx)
- Input System Package
- C#

### Controls

 W - Accelerate  
 S - Brake  
 A - Turn left  
 D - Turn right  

### How to Run

1. Clone the repository:
  ```bash
  git clone https://github.com/ezblk2-lgtm/UnitySimpleGame.git
  ```
2. Open the project in Unity Hub (Unity 6000.0.xx or newer)

3. Make sure the following packages are installed: Input System

4. Open the scene: `Assets/Scenes/MenuScene.unity`

5. Press Play or build the project via **File → Build Settings**
