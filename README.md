# Triffic Runner

Простая 3D игра в жанре "бесконечный бегун" на Unity. Игрок управляет машиной, движется по бесконечно генерируемой трассе и уворачивается от встречных автомобилей.

## Особенности

- Бесконечная генерация дороги с препятствиями
- Система счета (чем дальше — тем больше очков)
- Главное меню и меню настроек
- Настройка разрешения, полноэкранного режима и громкости
- Звук двигателя с изменением тона от скорости
- Сохранение настроек через PlayerPrefs

## Технологии

- Unity 6 (6000.0.xx)
- Input System Package
- C#

## Управление
 W - Ускорение\
 S - Торможение\ 
 A - Поворот влево\ 
 D - Поворот вправо\ 

## Запуск проекта

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ezblk2-lgtm/UnitySimpleGame.git

2. Откройте проект в Unity Hub (версия Unity 6000.0.xx или выше)

3. Убедитесь, что установлены пакеты: Input System

4. Откройте сцену Assets/Scenes/MenuScene.unity

5. Нажмите Play или соберите проект через File → Build Settings
